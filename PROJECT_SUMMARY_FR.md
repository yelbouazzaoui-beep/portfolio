→ Éditer `src/data/portfolio.ts` et committer

→ Créer nouveau composant dans `src/components/`
→ Importer dans `src/pages/index.astro`
→ Ajouter données dans `src/data/portfolio.ts`

### Pour customiser le design

→ Généralement dans `src/layouts/BaseLayout.astro` (variables)
→ Ou dans le composant spécifique (`src/components/*.astro`)
